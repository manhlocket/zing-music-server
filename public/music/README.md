# Thư mục nhạc riêng

Bỏ file MP3 của bạn vào thư mục này rồi sửa `catalog.json`.

Ví dụ:

```json
[
  {
    "id": "bao-thuc-minion",
    "title": "New Minion Wake Up",
    "artist": "Alarm",
    "file": "New Minion Wake Up-nhacchuong123.com.mp3",
    "keywords": ["bao thuc", "wake up", "minion"]
  }
]
```

Lưu ý: GitHub thường không nên chứa file nhạc bản quyền. Nếu file lớn, hãy dùng URL riêng hoặc upload lên Render/GitHub Releases/Cloudinary/R2.
